<text>
**Design Movement:** Architectural Futurism with Blueprint Precision

**Core Principles:**
1. **Structural Clarity:** Every element is intentionally placed as if designed by an architect—grids, sharp lines, and precise alignment create a sense of engineered perfection.
2. **Depth Through Layering:** Use stacked visual layers (backgrounds, content planes, overlays) to create a sense of infrastructure and depth, mimicking the "stack" concept from the brand hierarchy.
3. **Electric Energy:** Cyan (#00D4FF) is not just an accent—it's the visual representation of "automation energy" flowing through the system, used sparingly for maximum impact.
4. **Dark Foundation:** A near-black (#0A0A0A) base conveys authority, seriousness, and premium positioning, contrasting sharply with the electric cyan.

**Color Philosophy:**
- **Base:** #0A0A0A (Deep Black) — Authority, infrastructure, foundation
- **Primary Text:** #FFFFFF (Pure White) — Clarity, directness
- **Accent/Energy:** #00D4FF (Electric Cyan) — Automation, technology, the "spark"
- **Secondary/Structure:** #333333 (Dark Grey) — Grid lines, borders, architectural elements
- **Reasoning:** The palette is intentionally minimal to avoid distraction. The cyan is the only "color," making it impossible to ignore when used.

**Layout Paradigm:**
- **Asymmetric Grid System:** Avoid centered, symmetrical layouts. Use off-center hero sections, diagonal dividers, and staggered content blocks.
- **Blueprint Grid Background:** Subtle grid lines (like architectural blueprints) in the background to reinforce the "infrastructure" metaphor.
- **Layered Sections:** Each major section should feel like a distinct "layer" of the system, visually separated by diagonal cuts, overlays, or depth shifts.

**Signature Elements:**
1. **Corner Brackets:** Small, geometric corner brackets (like the slide deck) to frame key sections and reinforce the "blueprint" aesthetic.
2. **Diagonal Section Dividers:** Use CSS `clip-path` to create diagonal cuts between sections, breaking the monotony of horizontal layouts.
3. **Glowing Cyan Lines:** Subtle glow effects on cyan elements (borders, icons, CTAs) to create a sense of "energy flow."

**Interaction Philosophy:**
- **Micro-Animations:** Subtle entrance animations (fade-in, slide-up) for sections as they scroll into view.
- **Hover States:** Cyan glow intensifies on hover for CTAs and interactive elements.
- **Scroll-Triggered Effects:** Grid lines or background elements subtly animate as the user scrolls, creating a sense of depth and motion.

**Animation:**
- **Entrance:** Fade-in + slight upward motion for text and cards (0.6s ease-out).
- **Hover:** Cyan glow expansion on buttons and links (0.3s ease).
- **Scroll:** Parallax effect on background grid or hero elements (subtle, not distracting).

**Typography System:**
- **Display (Headings):** Inter Extra Bold (800) — Sharp, authoritative, uppercase for major headlines.
- **Body (Paragraphs):** Roboto Mono (400) — Technical, precise, monospaced for a "code/blueprint" feel.
- **Accent (Labels/Metadata):** Roboto Mono (700) — Uppercase, letter-spaced, for section labels and CTAs.
- **Hierarchy:**
  - H1: 64px (Inter 800, Uppercase, -2px letter-spacing)
  - H2: 48px (Inter 800, Uppercase, -1px letter-spacing)
  - H3: 32px (Inter 800, Uppercase)
  - Body: 18px (Roboto Mono 400, 1.6 line-height)
  - Label: 14px (Roboto Mono 700, Uppercase, 2px letter-spacing)

## Design Commitment

We are building an **Architectural Futurism** website that embodies the "infrastructure" positioning of CustomerAutomation.com. This is not a typical SaaS landing page—it's a visual manifesto that uses dark mode, electric cyan, and blueprint-inspired elements to convey authority, precision, and cutting-edge technology.

Every design decision will reinforce the core message: "This is not a tool. This is infrastructure."
